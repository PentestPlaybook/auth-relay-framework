1. Install F-Droid

a) Download F-Droid from: https://f-droid.org/en/
b) Double-click the APK file to install it

------------------------------------------------------------------------------------------------------------------------------------
2. Install Termux and X11:

a) Install "Termux (Terminal Emulator with packages)" from F-Droid App
b) Install Termux:X11 (Download from: https://github.com/termux/termux-x11/releases/download/nightly/app-arm64-v8a-debug.apk)
C) Double-click the Termux:X11 APK file to install it

------------------------------------------------------------------------------------------------------------------------------------
3. Enable SSH Access:

pkg update
pkg install openssh
passwd
sshd
whoami

------------------------------------------------------------------------------------------------------------------------------------
4. SSH Connect to Termux from your Host:

cd C:\Users\[YOUR WINDOWS USERNAME]\Downloads\platform-tools-latest-windows\platform-tools
.\adb forward tcp:8022 tcp:8022
ssh -p 8022 u0_a###@localhost

------------------------------------------------------------------------------------------------------------------------------------
5. Installations:

pkg install vim && pkg install wget && pkg install libandroid-support && pkg install libandroid-support && pkg install libandroid-support && pkg install libandroid-support && pkg install libandroid-posix-semaphore && pkg install x11-repo && pkg install termux-x11-nightly && pkg install geckodriver && pkg install sshpass && pip install selenium --break-system-packages

------------------------------------------------------------------------------------------------------------------------------------
6. Identify your Termux Username:

whoami

------------------------------------------------------------------------------------------------------------------------------------
7. Switch to Root and Grant Termux Permissions in Magisk:

a) Switch to root 
su

b) In Magisk, ensure root access is enabled for:
[SharedUID] Shell
[SharedUID] Termux

------------------------------------------------------------------------------------------------------------------------------------
